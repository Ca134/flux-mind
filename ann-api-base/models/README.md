# Private ANN Model

The trained ANN checkpoint used by the FluxMind backend is not included in this
public repository.

In the public demo architecture, the ANN model runs on the author's private
server. The open-source code keeps the API contract, client integration, and
deployment instructions so that the agent can call a compatible ANN service.

If you need access to the hosted ANN service or the private model checkpoint,
please contact the author to request permission and usage details.
